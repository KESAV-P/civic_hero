# How to Run Civic Hero from this Export

1.  **Extract the Zip File**: Unzip the contents of `CivicHero_Project_Export.zip` to a folder.
2.  **Open Terminal**: Open your terminal or command prompt in that folder.
3.  **Install Dependencies**: Run the following command to install all necessary libraries:
    ```bash
    npm install
    ```
4.  **Start the Development Server**:
    ```bash
    npm run dev
    ```
5.  **Open in Browser**: seamlessy use the app at [http://localhost:3000](http://localhost:3000).
